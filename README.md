# Lynq Case Landing (Next.js + HeroUI v3)

Minimal Apple-like landing page built with **Next.js (App Router)** and **HeroUI v3 React components**.

## Requirements

- Node.js + npm (HeroUI/Next requires Node to run)

## Setup

From this directory:

```bash
npm install
npm run dev
```

Then open:

- `http://localhost:3000`
